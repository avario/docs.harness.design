# Diode

A **diode** is an electrical component that allows current to flow in only one direction. Each diode can be assigned a [diode part](/parts/part-diode).

## Adding a Diode

Right-click on the canvas in the [Schematic View](/quick-start/schematic) and select **Diode**.  
<br />  
<img src="/components/diode-add.png" width={634} style={{ borderRadius: '8px' }} />

## Locating a Diode

Diodes are automatically positioned at branch points based on the most efficient wiring path. An automatically placed diode is shown with a dashed arrow line.  
<br />  
<img src="/components/diode-locate-automatic.png" width={634} style={{ borderRadius: '8px' }} />

You can manually position a diode by dragging its location handle in the [Layout View](/quick-start/layout) to any branch point, connector, or terminal in the harness.  
<br />  
<img src="/components/diode-locate-manual.png" width={634} style={{ borderRadius: '8px' }} />

<Callout>  
To reset a diode to automatic placement, select its location line and press **Delete**.  
</Callout>

## Flipping Diode Direction

Right-click a diode in the Schematic or Layout View and select **Flip direction** to reverse its orientation.  
<br />
