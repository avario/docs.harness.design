# Resistor

A **resistor** limits electrical current and reduces voltage in a circuit. Resistors can be assigned a [resistor part](/parts/part-resistor).

## Adding a Resistor

Right-click on the canvas in the [Schematic View](/quick-start/schematic) and select **Resistor**.  
<br />  
<img src="/components/resistor-add.png" width={634} style={{ borderRadius: '8px' }} />

## Locating a Resistor

Resistors are automatically positioned at branch points based on the most efficient wiring path. An automatically placed resistor is shown with a dashed arrow line.  
<br />  
<img src="/components/resistor-locate-automatic.png" width={634} style={{ borderRadius: '8px' }} />

You can manually position a resistor by dragging its location handle in the Layout View to any branch point, connector, or terminal in the harness.  
<br />  
<img src="/components/resistor-locate-manual.png" width={634} style={{ borderRadius: '8px' }} />

<Callout>  
To reset a resistor to automatic placement, select its location line and press **Delete**.  
</Callout>
