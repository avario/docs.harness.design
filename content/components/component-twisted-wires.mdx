# Twisted Wires

**Twisted wires** are a group of wires that should be twisted together when the harness is assembled. Usually this will be a twisted pair of two wires.

You can see which wires are twisted together in the **Connections View** by viewing the **Twisted With** column.

<br />
<img src="/components/twisted-wires.png" width={634} style={{ borderRadius: "8px" }} />

## Adding a Twisted Wires

Right-click on the canvas in the Schematic View and select **Twisted Wires** to add a new twisted wire group.

<br />
<img src="/components/twisted-wires-add.png" width={634} style={{ borderRadius: "8px" }} />

### Twist Existing Wires Together

Select multiple wires, right-click, and choose **Twist together**. This creates a twisted wire group with the selected wires.

<br />
<img src="/components/twisted-wires-twist-together.png" width={634} style={{ borderRadius: "8px" }} />
