# Splice
A splice is the connection of one or more wires within the harness.

## Add a Splice
Right click on the artboard of the Schematic view to add a Splice
<br />
<img src="/components/splice-add-right-click.png" width={634} style={{ borderRadius: '8px' }}/>
A Splice can be added by selecting multiple wires in the Schematic view and right clicking to select "Add Splice"
<br />
<img src="/components/splice-add-wires.png" width={634} style={{ borderRadius: '8px' }}/>

## Locate a Splice
Splice are located at branch points automatically in the harness based on the most efficent path for the wires. A Splice placed automatically is shown by a dashed arrow line
<br />
<img src="/components/splice-locate-automatic.png" width={634} style={{ borderRadius: '8px' }}/>

Splices can be located manually by dragging from the location bar on the splice in the Layout view to any branch point, connector or terminal in the harness.
<br />
<img src="/components/splice-locate-manual.png" width={634} style={{ borderRadius: '8px' }}/>
