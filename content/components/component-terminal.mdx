# Terminal

A <IconTerminal/> `Terminal` is a single electrical connection point and serves as a termination point in the harness. Each terminal can be assigned a [Terminal Part](/parts/part-terminal).

## Adding a Terminal

`right-click` on the canvas in the [Schematic](/quick-start/schematic) or [Layout](/quick-start/layout) view and go <IconTerminal/> `Terminal` and select the terminal type from the submenu.
<br />  
<img src="/components/terminal-add.png" width={368} style={{ borderRadius: '8px' }} />

## Editing Terminal Labels

`left-double-click` any terminal label to edit it.

## Adjust View Width of Terminal

If the name or cavity labels are too long, `right-click` the **Terminal**, select <IconArrowAutofitWidth/> `Width`, and set the desired width.
