# Terminal
A terminal is a single electrical connection point. It is a termination points of the harness

## Add a Terminal
Right click on the artboard on the Schematic or Layout View to add a Terminal
<br />
<img src="/components/terminal-add.png" width={634} style={{ borderRadius: '8px' }}/>

## Assign a [Terminal Part](/parts/part-terminal)
Right click on a Terminal in the Schematic or Layout View and select the Terminal Part you want to assign
<br />
<img src="/components/terminal-assign.png" width={320} style={{ borderRadius: '8px' }}/>

## Locate a Terminal
Drag a Terminal by clicking and dragging it in the Layout View
<br />
