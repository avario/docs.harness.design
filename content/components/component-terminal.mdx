# Terminal

A **terminal** is a single electrical connection point and serves as a termination point in the harness. Each terminal can be assigned a [terminal part](/parts/part-terminal).

## Adding a Terminal

Right-click on the canvas in the [Schematic](/quick-start/schematic) or [Layout](/quick-start/layout) view and select **Terminal**.
