# Terminal

A **terminal** is a single electrical connection point and serves as a termination point in the harness. Each terminal can be assigned a [terminal part](/parts/part-terminal).

## Adding a Terminal

Right-click on the canvas in the [Schematic](/quick-start/schematic) or [Layout](/quick-start/layout) view and select the terminal type from the submenu.  
<br />  
<img src="/components/terminal-add.png" width={368} style={{ borderRadius: '8px' }} />

## Editing Terminal Labels

Double-click any terminal label to edit it.

## Adjust View Width of Terminal

If the name or cavity labels are too long, right-click the terminal, select **Width**, and set the desired width.
