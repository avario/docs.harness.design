# Bundle
A bundle is used to represent a collection of wires. It can have dimensions and coverings applied to it

## Add a Bundle
In the Layout View, drag from one connection point of a Contact, Terminal or Branch Point to another
<br />
<img src="/components/bundle-add.png" width={634} style={{ borderRadius: '8px' }}/>

## Assign a [Covering Part](/parts/part-covering)
Right click on a Bundle in the Layout View and select the Covering Part you want to assign. Multiple coverings can be assigned to a single Bundle
<br />
<img src="/components/bundle-assign.png" width={634} style={{ borderRadius: '8px' }}/>

## Add dimensions
Select a bundle in the Layout View then double click the dimension and type in the value you want. Adding "mm", "cm", "m", "in", "ft" or "yrd" will automatically set the unit of measurement. Deffault length unit can be set under Preferences in the Main Menu
<br />
<img src="/components/bundle-dimension.png" width={634} style={{ borderRadius: '8px' }}/>
