# Bundle

A **bundle** represents a group of wires. It can have a defined length and optional [coverings](/parts/part-covering) applied.

Bundles can only be added in the [Layout View](/quick-start/layout).

## Adding Bundle

In the Layout View, drag from the connection point of a connector, terminal, or branch point to another.  
<br />  
<img src="/components/bundle-add.png" width={634} style={{ borderRadius: '8px' }} />

## Setting Bundle Lengths

Select a bundle, then double-click the input field to set its length.

You can enter units such as `mm`, `cm`, `m`, `in`, `ft`, or `yd`. The appropriate unit will be applied to the length.
