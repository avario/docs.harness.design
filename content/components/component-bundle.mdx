# Bundle

A **bundle** represents a group of wires. It can have a defined length and optional [coverings](/parts/part-covering) applied.

Bundles can only be added in the [Layout View](/quick-start/layout).

## Adding Bundle

In the Layout View, drag from the connection point of a connector, terminal, or branch point to another.  
<br />  
<img src="/components/bundle-add.png" width={634} style={{ borderRadius: '8px' }} />

## Setting Bundle Lengths

Select a bundle, then double-click the input field to set its length.

You can enter units such as `mm`, `cm`, `m`, `in`, `ft`, or `yd`. The appropriate unit will be applied to the length.

</Callout>

## Layout Points

Layout points can be added to **Bundles** to manually route them. This will override the automatic routing. To add a **Layout point**, right-click on the bundle in the **Layout View** and select **Add layout point**.  
<br />  
<img src="/components/bundle-layout.png" width={634} style={{ borderRadius: '8px' }} />
You can then drag the **Layout point** to adjust the bundle's path.
<br />  
<img src="/components/bundle-layout-drag.png" width={634} style={{ borderRadius: '8px' }} />

<Callout>  
More than one **Layout point** can be added to a bundle for more complex routing.
</Callout>

## Empty Bundles

A bundle with no wires running through it will show up with a dotted line.
