# Sharing

To let other people view your harness, go to <IconMenu2 /> `Main Menu` → <IconShare2 /> `Share`.

<br />
<img src="/view/main-menu/share.png" width={400} style={{ borderRadius: "8px" }} />

<Callout>
  Only the harness owner can edit the harness. Anyone the harness is shared with will only be able to view it in
  read-only mode.
</Callout>
