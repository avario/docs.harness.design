## How It Works

The information shown updates based on what is selected. The table below describes what to expect for each selection type.

| Selection | Information Shown |
|-----------|-------------------|
| Any single component | Component ID |
| Multiple components | Number of components selected |
| One or more wires | Total gauge; length (single wire only) |
| Single splice | Number of connected wires; total gauge |
| Single bundle | Total number of wires; total gauge |
| Single connector | Number of connected wires |
| Single terminal | Number of connected wires; total gauge |

### Parts

Clicking a part shows the following.

| Part | Information Shown |
|------|-------------------|
| Splice | Min/max gauge |
| Cable | Cores - Shield |
| Connector | Cavities - Shell - Gender |
| Contact | Crimp type - Gender - Min/max gauge |
| Terminal | Size - Min/max gauge |
| Covering | Min/max diameter - Fixed length |
