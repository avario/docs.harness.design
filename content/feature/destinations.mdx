# <IconLabel/>Destinations

With <IconLabel/>**Destinations** enabled, clicking any cavity or terminal on the Schematic shows a destination label. Type a connector or cavity name to search the harness and instantly create a connection.

## Enabling Destinations

To enable **Destinations**, click the button in the bottom right of the schematic view, or use the shortcut `D`.

<br />
<img src="/feature/destinations-enable.png" width={400} style={{ borderRadius: "8px" }} />

## Using Destinations

Select a component on the Schematic to reveal destination inputs next to each of its cavities. Click the input beside the cavity you want to connect, type a component or cavity name to filter results, then press `Enter` or `left-click` a result to instantly create a wire connection.
