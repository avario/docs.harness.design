# Schematic

The <IconManualGearbox/> `Schematic` view is a visual representation of all the electrical connections in your harness. In this view you can add **Connector**, **Terminal**, **Splice**, **Diode**, and **Resistor** [Components](/component) and connect them together with **Wires**, **Twisted Wires** or **Cables**. You can then assign specific [Parts](/part) to each component.

<br />
<img src="/view/schematic.png" width={800} style={{ borderRadius: "8px" }} />

## Canvas

This is where [Components](/component) are placed and connected together with [Wires](/component/wire).

`right-click` anywhere on the canvas to open the context menu and add [Components](/component).

`right-click` any component to assign a [Part](/part).

See [Keyboard Shortcuts & Mouse Gestures](/shortcuts) for how to navigate the **Schematic** view.

## View Options Menu Schematic
<br/>
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem"
  }}
>
  <li><IconNut/> `Show parts` - Toggles the visibility of assigned parts</li>
  <li><IconAlertHexagon/> `Unassigned part warnings` - Toggles warnings for components with unassigned parts</li>
  <li><IconHash/> `Show component IDs` - Toggles visibility of the unique ID on and off for all components. [More info](/component/splice#viewing-splice-component-ids)</li>
  <li><IconBolt/> `Show nets for selection` - Toggles between showing the complete path of conductance and showing what is directly connected to the selected component. [More info](/feature/nets#highlight-wires-attached-to-selected-part)</li>
  <li><IconLineDashed/> `Dash undefined routes` - Toggles dashed lines for wires that have not been routed through a Bundle in the Layout view. [More info](/component/wire#dashed-wires)</li>
  <li><IconOrthogonal/> `Auto layout all paths` - Automatically arranges all wires to be orthogonal (90-degree angles) [More info](/component/layout-point#auto-layout)</li>
  <li><IconMinus/> `Clear all layout points` -  Deletes all layout points from wires</li>
  <li><IconLock/> `Read-only mode` - Toggles read-only mode, which prevents any changes from being made to the harness</li> 
</ul>
