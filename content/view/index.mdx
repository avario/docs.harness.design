## Switching Views

`left-click` on the <IconChevronDown/> next to the view name in the top-right corner and select the view you want to switch to.

## Add Split Screen

To add a new view in split screen, `left-click` on the <IconLayoutSidebarRightFilled/> next to the view name in the top-right corner and select the view you would like to add. You can open as many views as you want in split screen.

## Remove Split Screen

To remove a view from split screen, `left-click` on the <IconX/> next to the view name in the top-right corner of the view you want to remove.
