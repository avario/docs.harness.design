# <IconSettings/>Settings

The <IconSettings/>**Settings** modal is organised into vertical tabs down the left side, split into two groups:

- **User settings** apply to your account.
- **Harness settings** apply only to the harness you currently have open.

<br />
<img src="/view/main-menu/settings.png" width={600} style={{ borderRadius: "8px" }} />

## User settings

### <IconCircleHalf2/>Appearance

#### Color scheme

The software supports both light and dark color schemes.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem",
  }}
>
  <li>
    <IconSun /> **Light** - Use the light color scheme
  </li>
  <li>
    <IconMoon /> **Dark** - Use the dark color scheme
  </li>
  <li>**Auto** - Match the color scheme to your system setting</li>
</ul>

#### Wire stripes

The **Show wire stripes as dashed lines** checkbox toggles how striped wires are drawn. When on, stripes are shown as dashes along the wire for easier reading at a glance. When off, stripes run parallel along the wire's length for a more realistic appearance.

### <IconMouse/>Input

#### Input type

Forces the software to interpret scrolling for a specific input device.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem",
  }}
>
  <li>
    <IconMouse /> **Mouse** - Scrolling zooms the canvas. Click and drag the middle mouse button to
    pan.
  </li>
  <li>
    <IconHandClick /> **Trackpad** - Scrolling pans the canvas. Pinch to zoom in and out.
  </li>
  <li>**Auto** - Automatically detect the input device</li>
</ul>

## Harness settings

These tabs only affect the harness you currently have open, and are hidden while the harness is read-only.

### <IconLock/>Read-only

Toggle **Make this harness read-only** to disable all editing of the harness.

### <IconRuler2/>Units

Sets the currency symbol used throughout the harness.

### <IconChalkboardTeacher/>Formboard

The **Formboard** tab turns the layout view into a 1:1 physical scale drawing of your harness. It's an Enterprise feature with its own settings for panel size, grid, bend radius, and scale. [Read more...](/feature/formboard)
