# <IconSettings/> Settings

The <IconSettings/> `Settings` panel allows you to set default global settings for the software.

<br />
<img src="/view/main-menu/settings.png" width={400} style={{ borderRadius: "8px" }} />

## Default length unit

This sets the default length unit used throughout the software.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem",
  }}
>
  <li>`mm` - Millimeter</li>
  <li>`cm` - Centimeter</li>
  <li>`m` - Meter</li>
  <li>`in` - Inch</li>
  <li>`ft` - Foot</li>
  <li>`yd` - Yard</li>
</ul>

## Default gauge unit

This sets the default gauge unit used throughout the software. For metric `mm²` is used, while for imperial `AWG` is used.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem",
  }}
>
  <li>`Metric` - Millimeters squared (mm²)</li>
  <li>`AWG` - American Wire Gauge</li>
</ul>

## Input type

This allows you to force the software to a certain input type.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem",
  }}
>
  <li>
    <IconMouse /> **Mouse** - Scrolling will zoom the canvas. Click and drag the middle mouse button
    to pan.
  </li>
  <li>
    <IconHandClick /> **Trackpad** - Scrolling will pan the canvas. Pinch to zoom in and out.
  </li>
  <li>**Auto** - Automatically detect the input device</li>
</ul>

## Color scheme

The software supports both light and dark color schemes.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem",
  }}
>
  <li>
    <IconSun /> **Light** - Use light color scheme
  </li>
  <li>
    <IconMoon /> **Dark** - Use dark color scheme
  </li>
  <li>**Auto** - Set light or dark color schemes based on the system settings</li>
</ul>
