# <IconFileExport/>Export

<IconFileExport/>`Export` allows you to output documents for manufacturing your harnesses or to
save a local copy of your harness data.

<br />
<img src="/quick-start/export.png" width={200} style={{ borderRadius: "8px" }} />

## <IconFile/>Schematic

Exports a `PDF` version of the [Schematic](/view/schematic) view.

## <IconFile/>Layout

Exports a `PDF` version of the [Layout](/view/layout) view.

## <IconTable/>Wiring table

Exports an Excel `.xlsx` file based on the [Connections](/view/connections) view. The file contains a complete list of all connections, followed by individual sheets that break down connections per connector.

<br />
<img src="/view/main-menu/export-xlsx.png" width={800} style={{ borderRadius: "8px" }} />

## <IconTable/>BOM

Exports a `CSV` version of the [Parts](/view/parts) view.

## <IconBraces/>Raw data

Exports a `JSON` file containing all the data of the harness to local storage.

Click here to see how to [open local data](/view/main-menu/open#local-data)
