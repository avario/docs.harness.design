# <IconMenu2/>Main Menu

Click the <IconMenu2/>**Main Menu** icon in the top left corner.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem"
  }}
>
  <li><IconFile/>**New harness** - Creates a new harness under your account</li>
  <li><IconFolder/>**Open** - [Opens](/view/main-menu/open) the Open Harness panel</li>
  <li><IconFileExport/>**Exports** - [Exports](/view/main-menu/export) the current harness</li>
  <li><IconShare2/>**Share** - [Share](/view/main-menu/share) opens the Share panel</li>
  <li><IconHistory/>**Revisions** - [Revisions](/feature/revisions) opens the revision history</li>
  <li><IconSettings/>**Harness settings** - [Harness Settings](/view/main-menu/harness-settings) for the current harness (units, sharing, formboard, team, delete)</li>
  <li><IconPackage/>**Parts library** - [Opens](/view/parts/library) the parts library</li>
  <li><IconBook2/>**Guides** - [Guides](/) opens this documentation</li>
  <li><IconBrandDiscord/>**Discord** - [Discord](https://discord.com/invite/eTv6ph4Uzg) opens invite to discord server</li>
  <li><IconMessageReport/>**Give feedback** - Opens the feedback modal</li>
  <li><IconAdjustments/>**Preferences** - [Preferences](/view/main-menu/preferences) for your account (appearance, input, hotkeys)</li>
  <li><IconUserCircle/>**Account** - [Account](/view/main-menu/account) settings — profile, password, subscription, team, and log out</li>
  <li><IconPro/>**Upgrade to Pro** - Opens the upgrade plan screen. See [Plans](/pro-plan) for more details</li>
</ul>
