# Main Menu

Click the <IconMenu2/> `Main Menu` icon in the top left corner.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem"
  }}
>
  <li><IconFile/> `New harness` - Creates a new harness under your account</li>
  <li><IconFolder/> `Open` - [Opens](/view/main-menu/open) the Open Harness panel</li>
  <li><IconFiles/> `Duplicate` - Duplicates the current harness</li>
  <li><IconFileExport/> `Exports` - [Exports](/view/schematic) the current harness</li>
  <li><IconShare2/> `Share` - [Share](/view/schematic) opens the Share panel</li>
  <li><IconTrash/> `Delete` - Deletes the current harness</li>
  <li><IconBook2/> `Guides` - [Guides](/) opens this documentation</li>
  <li><IconBrandDiscord/> `Discord` - [Discord](https://discord.com/invite/eTv6ph4Uzg) opens invite to discord server</li>
  <li><IconMessageReport/> `Give feedback` - Opens the feedback modal</li>
  <li><IconSettings/> `Settings` - [Settings](/view/main-menu/settings) Opens the Settings panel</li>
  <li><IconUserCircle/> `Account` - [Account](/view/schematic) opens the Account window</li>
  <li><IconLogout/> `Log out` - Logs out of the current account</li>
</ul>
