# <IconSettings/>Harness Settings

Open <IconMenu2/>**Main Menu** → <IconSettings/>**Harness settings**. These settings apply only to the **harness you currently have open**.

## <IconLock/>Read-only

Toggle **Make this harness read-only** to disable all editing of the harness. A read-only harness can still be viewed and exported, and you can switch editing back on from the same tab.

## <IconRuler2/>Units

#### Currency

Sets the currency symbol used throughout the harness.

#### Default length unit

Sets the default length unit used in the harness.

#### Default gauge unit

Sets the default gauge unit used in the harness.

## <IconShare2/>Sharing

The **Sharing** tab is the same as the [Share](/view/main-menu/share) dialog — make the harness public or invite people to it by email. See [Sharing](/feature/sharing) for the full guide.

## <IconChalkboardTeacher/>Formboard

The **Formboard** tab turns the layout view into a 1:1 physical scale drawing of your harness. It's an Enterprise feature with its own settings for panel size, grid, bend radius, and scale. [Read more...](/feature/formboard)

## <IconUsersGroup/>Team

Shown when you belong to a [team](/feature/teams). Use **Move to team** to share the harness with your team so seated members can edit it alongside you, or **Remove from team** to return it to personal ownership. See [Collaborating on harnesses](/feature/teams/collaborating).

## <IconTrash/>Delete

The **Delete** button at the bottom of the tab list permanently deletes the harness. This can't be undone.
