# <IconBrandStackshare/>Layout

The <IconBrandStackshare/>**Layout** view is a visual representation of the physical layout of the harness. In this view you can add **Connector**, **Terminal** and **Branch Point** [Components](/component) and connect them together with **Bundles**. You can then assign specific [Parts](/part) to each component such as **Coverings**.

<br />
<img src="/view/layout.png" width={800} style={{ borderRadius: "8px" }} />

## Canvas

This is where [Components](/component) are placed and connected together with [Bundles](/component/bundle).

`right-click` anywhere on the canvas to open the context menu and add [Components](/component).

`right-click` any component to assign a [Part](/part).
