# <IconCircuitResistor/>Resistor

A <IconCircuitResistor/>`Resistor` limits electrical current and reduces voltage in a circuit. Resistors can be assigned a [resistor part](/part/resistor).

## Adding a Resistor

`right-click` on the canvas in the [Schematic](/quick-start/schematic) view and select <IconCircuitResistor/>`Resistor`.  
<br />  
<img src="/component/resistor-add.png" width={634} style={{ borderRadius: '8px' }} />

## Locating a Resistor

Resistors are located on the **Layout** view the same way as [Splices](/component/splice#locating-a-splice).

## View Resistor Component IDs

Viewing the component ID of a resistor on the **Layout** and **Schematic** is done the same way as for a [Splice](/component/splice#viewing-splice-component-ids).
