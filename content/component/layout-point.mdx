# <IconPlus/>Layout Point

<IconPlus/> **Layout Points** are used to define custom layouts for [Wires](/component/wire), and [Cables](/component/cable) in the [Schematic](/view/schematic) View, and for [Bundles](/component/bundle) in the [Layout](/view/layout) View.

## Adding a Layout Point

`Double-click` a [Wire](/component/wire), [Cable](/component/cable), or [Bundle](/component/bundle) to add a [Layout Point](/component/layout-point). You can also `right-click` the item and select <IconPlus/>`Add layout point` from the context menu.

<Callout>
  More than one [Layout Points](/component/layout-point) can be added to a wire for more complex
  routing.
</Callout>

## Moving Layout Points

To move a [Layout Point](/component/layout-point), `left-click` and `drag` it to the desired position.

<br />
<img src="/component/wire-layout-drag.png" width={634} style={{ borderRadius: "8px" }} />

<Callout>
  You can `shift` + `left-click` to select, or marquee select, to move multiple **Layout Points** at
  once.
</Callout>

<Callout>
  When moving a [Layout Point](/component/layout-point), parallel guide lines will appear on the
  sides of the item being adjusted when it is aligned `horizontally`, `vertically`, or at `45°`.
</Callout>

`Left-click` and `drag` on a [Wire](/component/wire) or [Cable](/component/cable) will move the two layout points on either side of the dragged position.

<br />
<img src="/component/layout-point-drag-wire.png" width={634} style={{ borderRadius: "8px" }} />

## Deleting a Layout Point

- `Double-click` the [Layout Point](/component/layout-point)
- Select it and hit the `Delete` key
- `Right-click` and select <IconTrash/>`Delete`

## Deleting all Layout Points on a Wire, Cable or Bundle

`Right-click` a [Wire](/component/wire), [Cable](/component/cable), or [Bundle](/component/bundle) and select <IconMinus/>`Delete all layout points` from the context menu.

## Deleting All Layout Points in a View

In the bottom right of the [Schematic](/view/schematic) or [Layout](/view/layout) Views go <IconEye/>`View Options` → <IconMinus/>`Clear all layout points`. This will delete all layout points on the corresponding view.

## Orthogonal Layout

`Right-click` a [Wire](/component/wire) and select <IconOrthogonal/>`Make orthogonal` from the context menu or press the `O` key. This will layout these items orthogonally (only horizontal and vertical) using [Layout Points](/component/layout-point).

## Make All Paths Orthogonal

In the bottom right of the [Schematic](/view/schematic) View go <IconEye/>`View Options` → <IconOrthogonal/>`Make all paths orthogonal` or press `shift` `O`. This will lay out all [Wires](/component/wire) orthogonally. This will override any existing layout points.
