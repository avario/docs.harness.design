# Diode

A <IconCircuitDiode/> `Diode` is an electrical component that allows current to flow in only one direction. Each diode can be assigned a [diode part](/part/diode).

## Adding a Diode

`right-click` on the canvas in the [Schematic](/quick-start/schematic) view and select <IconCircuitDiode/> `Diode`.  
<br />  
<img src="/component/diode-add.png" width={634} style={{ borderRadius: '8px' }} />

## Locating a Diode

Diodes are located on the **Layout** view the same way as [Splices](/component/splice#locating-a-splice).

## Flipping Diode Direction

`right-click` a diode in the Schematic or Layout View and select <IconArrowsRightLeft/> `Flip direction` to reverse its orientation.  
<br />  
<img src="/component/diode-flip.png" width={634} style={{ borderRadius: '8px' }} />

## View Diode Component IDs

Viewing the component ID of a diode on the **Layout** and **Schematic** is done the same way as for a [Splice](/component/splice#viewing-splice-component-ids).
