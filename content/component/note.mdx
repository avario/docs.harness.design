# <IconNote/> Note

A <IconNote/> `Note` can be added to either the [Schematic](/quick-start/schematic) or [Layout](/quick-start/layout) views to add additional information.

## Adding a Note

`right-click` on the canvas in the [Schematic](/quick-start/schematic) or [Layout](/quick-start/layout) view and select <IconNote/> `Note`. `left-double-click` to enter text into the **Note**.

## Adjusting Note Color

`right-click` the **Note**, select `Color`, and select the desired color from the context menu.

## Adjusting Note Width

`right-click` the **Note**, select <IconArrowAutofitWidth/> `Width`, and select the desired width `%`.

## Adjusting Text Alignment

`right-click` the **Note**, select the alignment:

- <IconAlignLeft /> Left Justify
- <IconAlignCenter /> Center Justify
- <IconAlignRight /> Right Justify
- <IconLayoutAlignTop /> Align Top
- <IconLayoutAlignMiddle /> Align Center
- <IconLayoutAlignBottom /> Align Right
