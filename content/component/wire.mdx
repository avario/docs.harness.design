# Wire

A <IconMinus/> `Wire` represents an electrical connection between two components in the harness. Each wire can be assigned a [wire part](/part/wire).

## Adding a Wire

In the [Schematic View](/quick-start/schematic), `left-click-drag` from the connection handle of one component to another to create a **Wire**.

## Assigning Color

If no part is assigned to a **Wire**, you can manually set its color. `right-click` the **Wire** in the [Schematic](/view/schematic), [Layout](/view/layout), or [Conection](/view/connections) view and choose a **Color** from the `Color` menu. More colors can be selected by clicking `More` at the bottom of the color list. The **Stripe** color can be assigned in the same way.

<br />
<img src="/component/wire-color.png" width={634} style={{ borderRadius: "8px" }} />

<Callout>
  Wire stripe style can be changed under <IconMenu2/>`Main Menu` → <IconSettings/>`Settings`. [Click here](/view/main-menu/settings#wire-stripes) for more information
</Callout>

## Assigning Idents

To add a color **Ident** band to a **Wire**, `right-click` the wire and then click the <IconPlus/> button next to `Ident`. You can add multiple color **Ident** bands to a **Wire** and they will be displayed in the order they were added. To remove a color band, `right-click` on the wire and then click the color square for that **Ident**.

<br />
<img
  src="/component/wire-ident.png"
  width="40%"
  style={{ borderRadius: "8px", minWidth: "300px" }}
/>
<br />

To add a label ident to a **Wire**, `right-click` the **Wire** and then click on the **Ident** placeholder text in the menu.

The label background color can be set by adding a color ident with the <IconPlus/> button. If the number of color idents matches the number of characters in the label, each character will be displayed with a background color corresponding to the color ident.

<br />

## Adjusting the Wire Path

You can set a custom path for a **Wire** using [Layout Points](/component/layout-point).

## Dashed Wires

**Wires** can appear as dashed lines in the [Schematic](/view/schematic) view when they are not yet routed through a [Bundle](/component/bundle) in the [Layout](/view/layout) view.

<br />
<img src="/component/wire-dashed.png" width={634} style={{ borderRadius: "8px" }} />

<Callout>
  To hide dashed lines deselect <IconLineDashed /> `Dash undefined routes` in the <IconEye /> `View
  options` menu in the bottom right corner of the **Schematic** view.
  <img src="/component/wire-dashed-hide.png" width={326} style={{ borderRadius: "8px" }} />
</Callout>

## Jumper Wire

A `Jumper Wire` (loopback wire) can be created by `left-click-drag` from one cavity to another within the same connector in the **Schematic View**.

<br />
<img src="/component/wire-jumper.png" width={326} style={{ borderRadius: "8px" }} />

<Callout>
  A **Jumper Wire** is considered to have a length of **zero**. You may need to account for this
  when generating length reports.
</Callout>
