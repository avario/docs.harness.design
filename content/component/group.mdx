# <IconMaximize/>Group

A <IconMaximize/>`Group` contains a selection of components. It is commonly used to organise connectors that plug into a device (i.e. an ECU, fuse box etc). A group can be assigned a [Generic Part](/part/generic).

Groups can only be used in the [Schematic View](/view/schematic).

## Creating a Group

Select multiple components using `shift` `left-click` or `left-click-drag` selection, then `right-click` and select <IconMaximize/>`Create Group` to form a new group.

## Removing Components from a Group

`right-click` on the component you want to remove and select <IconMaximizeOff/>`Remove from group`.

## Wires in a Group

If a [Wire](/component/wire) connects two [Components](/component) that are in the same group, this **Wire** is considered to be part of that group. The group the wire is in can be seen in the [Connections](/view/connections) view under Group.
