# Terminal

A <IconTerminal/> `Terminal` is a single electrical connection point and serves as a termination point in the harness. Each terminal can be assigned a [Terminal Part](/part/terminal).

The following are the supported terminal types:

- <IconFerrule /> `Ferrule`
- <IconRing /> `Ring`
- <IconSpade /> `Spade`
- <IconMaleQuickConnect /> `Male Quick Connect`
- <IconFemaleQuickConnect /> `Female Quick Connect`
- <IconLooseWireEnd /> `Loose Wire End`

## Adding a Terminal

`right-click` on the canvas in the [Schematic](/quick-start/schematic) or [Layout](/quick-start/layout) view then use the <IconTerminal/> `Terminal` submenu to select the terminal type.

<br />
<img src="/component/terminal-add.png" width={368} style={{ borderRadius: "8px" }} />

## Editing Terminal Signals

`left-double-click` any terminal signal to edit it.

## Adjusting the Width of a Terminal

If the signal label is too long, `right-click` the **Terminal**, select <IconArrowAutofitWidth/> `Width`, and set the desired width.

<br />
<img src="/component/terminal-width.png" width={426} style={{ borderRadius: "8px" }} />

## Rotating a Terminal

**Terminals** can be rotated 90° in the **Schematic** view to improve complex layouts. To rotate , `right-click` a **Terminal** and select <IconRotate/> `Rotate`.

## Mating a Terminal

To mate a **Terminal** to a connector or another terminal see [Mates](/feature/mates).
