# Bundle

A `Bundle` represents a group of wires. It can have a defined length and optional [coverings](/part/covering) applied.

Bundles can only be added in the [Layout View](/quick-start/layout).

## Adding Bundle

In the Layout View, `left-click-drag` from the connection point of a connector, terminal, or branch point to another.  
<br />  
<img src="/component/bundle-add.png" width={634} style={{ borderRadius: '8px' }} />

## Setting Bundle Lengths

Select a bundle, then `left-double-click` the input field to set its length.

You can enter units such as `mm`, `cm`, `m`, `in`, `ft`, or `yd`. The appropriate unit will be applied to the length.

<Callout>
The default length unit can be set under **Main Menu → Preferences**.
</Callout>

<br />  
<img src="/component/bundle-dimension.png" width={634} style={{ borderRadius: '8px' }} />

## Layout a Bundle
To layout a **Bundle** you use [Layout Points](/component/layout-point).

## Empty Bundles

A bundle with no wires running through it will show up with a dotted line.
