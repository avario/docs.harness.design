# <IconMinus/>Bundle

A <IconMinus/>`Bundle` represents a group of wires. It can have a defined length and assigned [coverings](/part/covering).

Bundles can only be added in the [Layout View](/quick-start/layout).

## Adding Bundle

`left-click-drag` from the connection point of a [Connector](/component/connector), [Terminal](/component/terminal), or [Branch Point](/component/branch-point) to another.

## Setting Bundle Lengths

Select a bundle, then `left-double-click` the input field to set its length.

<br />
<video
  src="/quick-start/add-dimension.mp4"
  width={634}
  style={{ borderRadius: "8px" }}
  autoPlay
  loop
  muted
  playsInline
/>

<Callout>
For length, you can enter the following units (set the default unit under <IconMenu2/>`Main Menu` → <IconSettings/>`Settings` → <IconFile/>`Harness settings`):
  - `mm` - Milimeter
  - `cm` - Centimeter
  - `m` - Meter
  - `in` `"` - Inch
  - `ft` `'` - Foot
  - `yd` - Yard
</Callout>

## Setting the Bundle Path

You can use [Layout Points](/component/layout-point) to adjust the path of a **Bundle**.

## Bundle Label

`right-click` a **Bundle** and type the label name in the <IconLabel/>`Label` box. `left-click` the <IconColorSwatch/> button to select a color.

## Dashed Bundle Lines

A bundle with no wires running through it will be shown as a dashed line if <IconLineDashed/>`Dash undefined routes` is enabled in the <IconEye/>`View
  options`.
