# Cable

A <IconCable/> `Cable` is a group of individual wires—called **cores**—wrapped in a sheath.

[Cable parts](/part/cable) can be assigned to cables created in the [Schematic View](/quick-start/schematic).

## Adding a Cable

`right-click` on the canvas in the Schematic View and select <IconCable/> `Cable` to add a new cable.  
<br />  
<img src="/component/cable-add.png" width={634} style={{ borderRadius: '8px' }} />

### Create a Cable from Wires

Select multiple wires, `right-click`, and choose <IconCable/> `Make cable`. This creates a cable with a core for each selected wire.  
<br />  
<img src="/component/cable-add-wires.png" width={634} style={{ borderRadius: '8px' }} />

## Creating Cable Connections

`left-click-drag` from a component's connection point to one of the cable's cores. Once two components are connected to the same core, a connection is established using that core as the conductor.

## Layout a Cable
To layout a **Cable** you use [Layout Points](/component/layout-point).

## Cable Routing

Cables are routed automatically based on connections and bundles. The cable length is determined by its longest core.  
<br />  
<img src="/component/cable-route.png" width={800} style={{ borderRadius: '8px' }} />

## Adding and Removing Cable Cores

Select the cable in the Schematic View, then click the <IconPlus/> button to add cores, or the <IconMinus/> button to remove them. Enter the desired number of cores directly into the core count box.
<br />  
<img src="/component/cable-add-wire.png" width={141} style={{ borderRadius: '8px' }} />

## Adding a Shield to a Cable

`right-click` the cable in the Schematic View and select <IconShield/> `Add Shield`.  
<br />  
<img src="/component/cable-sheild.png" width={634} style={{ borderRadius: '8px' }} />

Shields can be connected to components just like the other wires in the cable.  
<br />  
<img src="/component/cable-shield-example.png" width={634} style={{ borderRadius: '8px' }} />
