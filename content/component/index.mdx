# Components

In both the [Schematic](/view/schematic) and [Layout](/view/layout) views, your harness is built using **Components**.

Each component can later be assigned a specific [Part](/part) from the [Parts](/view/parts) view. Once assigned, the component is validated against the properties of that part.

This workflow allows you to design your schematic and layout using generic components first, then assign actual parts when ready.
