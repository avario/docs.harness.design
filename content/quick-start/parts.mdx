# Create and Assign Parts

<Steps>
### Open the Parts View  
Click the view dropdown and select **Parts**.  
<br />  
<img src="/quick-start/parts-view-open.png" width={256} style={{ borderRadius: '8px' }} />

### Add [Parts](/parts)  
Right-click on the Parts view, then select a part from the dropdown.  
<br />  
<img src="/quick-start/right-add-part.png" width={147} style={{ borderRadius: '8px' }} />

### Edit Part Properties  
Click the chevron on the right side of a part row to adjust its properties.  
<br />  
<img src="/quick-start/edit-part.png" width={634} style={{ borderRadius: '8px' }} />

### Assign Parts  
Right-click on a component in the Schematic or Layout view and select the part you want to assign.  
<br />  
<img src="/quick-start/assign-part.png" width={634} style={{ borderRadius: '8px' }} />
<Callout>
  The quantities shown in the Parts view will be automatically updated as you assign parts.
</Callout>
</Steps>
