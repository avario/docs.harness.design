# Create and Assign Parts

<Steps>
### Open the <IconNut/> [Parts](/view/parts) View  
Click the view dropdown and select **Parts**.  
<br />  
<img src="/quick-start/parts-view-open.png" width={256} style={{ borderRadius: '8px' }} />

### Add [Parts](/part)  
In the Parts View, `right-click` on the canvas or click the <IconPlus/> `Add part` button in the bottom-left corner to add a new [Part](/part) .

<br />
<img src="/part/part-add.png" width={222} style={{ borderRadius: "8px" }} />

### Edit Part Properties  
`left-double-click` on a part name or click the <IconEdit /> button to view and edit a part's properties.

<br />
<img src="/part/part-edit.png" width={634} style={{ borderRadius: "8px" }} />

### Assign Parts  
`right-click` on a component in the [Schematic](/view/schematic) or [Layout](/view/layout) view and select the part you want to assign. The quantities shown in the [Parts](/view/parts) View will be automatically updated as you assign parts.
<br />  
<img src="/quick-start/assign-part.png" width={634} style={{ borderRadius: '8px' }} />
</Steps>

<Callout>
  Part properties can be shown or hidden. [More info](/view/parts#view-options-menu-parts).
</Callout>
