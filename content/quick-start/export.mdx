# Export

<Steps>
### Export Wiring Table and BOM  
Go to **Main Menu â†’ Export** and select **Wiring Table** or **BOM** to export as a `.csv` file. This can be opened in any spreadsheet application.

### Export Layout  
Click the download button at the bottom right of the Layout view to export the layout as a `.png`.  
<br />  
<img src="/quick-start/layout-export.png" width={216} style={{ borderRadius: '8px' }} />

<Callout>  
Enable **Show covering part details** in the view options to include them in the export.  
<br />  
<img src="/quick-start/layout-export-show.png" width={311} style={{ borderRadius: '8px' }} />
</Callout>

### Export Schematic  
Click the download button at the bottom right of the Schematic view to export the schematic as a `.png`.  
<br />  
<img src="/quick-start/schematic-export.png" width={216} style={{ borderRadius: '8px' }} />

<Callout>  
Enable **Always show wire parts** in the view options to include them in the export.  
<br />  
<img src="/quick-start/schematic-export-show.png" width={311} style={{ borderRadius: '8px' }} />
</Callout>
</Steps>
