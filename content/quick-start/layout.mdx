# Layout your Harness

<Steps>
### Open the <IconBrandStackshare/>[Layout](/view/layout) View  
Click the view dropdown and select **Layout**.  
<br />  
<img src="/quick-start/layout-view-open.png" width={256} style={{ borderRadius: '8px' }} />

### Add [Bundles](/component/bundle)  
`left-click-drag` from the connection point of a [Connector](/component/connector), [Terminal](/component/terminal), or [Branch Point](/component/branch-point) to another.
<Callout>
  Wires from your schematic will be automatically routed using the shortest bundle path.
</Callout>
<br />  
<video src="/quick-start/add-bundle.mp4" width={634} style={{ borderRadius: '8px' }} autoPlay loop muted playsInline />
<Callout>
  [Layout Points](/component/layout-point) can be added to **Bundles** in the same way as a **Wires**.
</Callout>

### Add [Branch Points](/component/branch-point)  
`right-click` on a [Bundle](/component/bundle) and select <IconArrowsSplit2/>`Add Branch Point` from the context menu.  
<br />
<video src="/quick-start/add-branch-point.mp4" width={634} style={{ borderRadius: '8px' }} autoPlay loop muted playsInline />

### Locate Internal Components
[Splices](/component/splice), [Diodes](/component/diode), and [Resistors](/component/resistor) are automatically positioned to minimise wire lengths. To place them manually, `left-click-drag` from a location handle to a specific branch point or component.
<br />  
<video src="/quick-start/locate-splice.mp4" width={634} style={{ borderRadius: '8px' }} autoPlay loop muted playsInline />

### Set Bundle Lengths
Select a [Bundle](/component/bundle), then `left-double-click` the input field to set its length.
<br />
<video src="/quick-start/add-dimension.mp4" width={634} style={{ borderRadius: '8px' }} autoPlay loop muted playsInline />
</Steps>
