# Layout your Harness

<Steps>
### Open the Layout view
Click the view drop down to change to the Layout view
<br />
<img src="/quick-start/layout-view-open.png" width={256} style={{ borderRadius: '8px' }}/>

### Add a [Bundle](/components/component-bundle)
Drag from the connection point of one connector, terminal or branch point to another
<br />
<img src="/quick-start/add-bundel.png" width={634} style={{ borderRadius: '8px' }}/>

### Add a [Branch Point](/components/component-baranch-point)
 Right clicking on a bundle, multiple bundles or the artboard and select Add Branch Point from the dropdown
 <br />
<img src="/quick-start/add-branch-point.png" width={634} style={{ borderRadius: '8px' }}/>
<br />

### Locate internal components
 Locate Splices, Diodes and Resistors by dragging the location arrow to a connector, terminal or branch point
 <br />
<img src="/quick-start/locate-splice.png" width={634} style={{ borderRadius: '8px' }}/>
<br />

### Add dimensions
 Select a bundle then double click to add a dimension
 <br />
<img src="/quick-start/add-dimensions.png" width={634} style={{ borderRadius: '8px' }}/>
<br />

</Steps>
