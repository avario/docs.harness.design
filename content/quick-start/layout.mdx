import { Steps } from 'nextra/components'
import { Callout } from 'nextra/components'
import { IconBrandStackshare, IconArrowsSplit2 } from "@tabler/icons-react";
 
# Layout your Harness

<Steps>
### Open the <IconBrandStackshare/> [Layout](/views/view-layout) View  
Click the view dropdown and select **Layout**.  
<br />  
<img src="/quick-start/layout-view-open.png" width={256} style={{ borderRadius: '8px' }} />

### Add [Bundles](/components/component-bundle)  
`left-click` `drag` from the connection point of a [Connector](/components/component-connector), [Terminal](/components/component-terminal), or [Branch Point](/components/component-branch-point) to another.
<Callout>
  Wires from your schematic will be automatically routed using the shortest bundle path.
</Callout>
<br />  
<img src="/quick-start/add-bundle.png" width={634} style={{ borderRadius: '8px' }} />
<Callout>
  [Layout Points](/components/component-layout-point) can be added to **Bundles** in the same way as a **Wires**.
</Callout>

### Add [Branch Points](/components/component-baranch-point)  
`right-click` on a [Bundle](/components/component-bundle) and select <IconArrowsSplit2/> `Add Branch Point` from the context menu.  
<br />
<img src="/quick-start/add-branch-point.png" width={634} style={{ borderRadius: '8px' }} />

### Locate Internal Components
[Splices](/components/component-splice), [Diodes](/components/component-diode), and [Resistors](/components/component-resistor) are automatically positioned to minimise wire lengths. To place them manually, `left-click` `drag` from a location handle to a specific branch point or component.  
<br />  
<img src="/quick-start/locate-splice.png" width={634} style={{ borderRadius: '8px' }} />

### Set Bundle Lengths
Select a [Bundle](/components/component-bundle), then `left-double-click` the input field to set its length.  
<br />
<img src="/quick-start/add-dimensions.png" width={634} style={{ borderRadius: '8px' }} />
</Steps>
