# Design your Schematic

<Steps>
### Open the Schematic View  
Click the view dropdown and select **Schematic**.  
<br />  
<img src="/quick-start/schematic-view-open.png" width={256} style={{ borderRadius: '8px' }} />

### Add [Components](/components)  
Right-click on the canvas and choose a component from the dropdown menu.  
<br />  
<img src="/quick-start/right-add-component.png" width={147} style={{ borderRadius: '8px' }} />

### Add [Wires](/components/component-wire)  
Drag from a connection handle on one component to another.  
<br />
<img src="/quick-start/add-wire.png" width={634} style={{ borderRadius: '8px' }} />
</Steps>
