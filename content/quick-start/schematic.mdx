# Design your Schematic

<Steps>
### Open the Schematic view
Click the view drop down to change to the Schematic view
<br />
<img src="/quick-start/schematic-view-open.png" width={256} style={{ borderRadius: '8px' }}/>

### Add a [Component](/components)
Right click and select a component from the dropdown menu
<br />
<img src="/quick-start/right-add-component.png" width={147} style={{ borderRadius: '8px' }}/>

### Add a [Wire](/components/component-wire)
Drag from the connection handle of one component to another component
<br />
<img src="/quick-start/add-wire.png" width={634} style={{ borderRadius: '8px' }}/>

### Add a [Splice](/components/component-splice) 
Right click on a wire, multiple wires or the artboard and select Add splice from the dropdown
<br />
<img src="/quick-start/add-splice.png" width={634} style={{ borderRadius: '8px' }}/>

</Steps>
