# April 20, 2026

This release introduces wire stripe display modes, document-level read-only settings, a student discount for the Pro Plan, notes in PDF exports, and signals in the wiring table.

### Wire Stripe Display

The visual style of striped wires can be toggled between two display modes: parallel stripes (which follow the wire's length for a more realistic appearance) and perpendicular dashes (which cross the wire for improved readability at a glance).

### Document Level Read-only

Documents can be set to read-only. This is saved to the document settings so that it cannot be accidentally changed.

### Student Discount

Students can now apply for a discounted Pro Plan subscription. Applications require a university email address.
