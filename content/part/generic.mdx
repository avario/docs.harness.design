# Generic Part
A generic part represents something the harness is connected to that you want tracked in the BOM. Generic parts can only be assigned to [Groups](/component/group).

## Assigning a Generic Part

`right-click` on the heading of a [Group](/component/group) in the Schematic or Layout View and select the `Part` you want to assign.
