# Parts

The <IconNut/> `Parts` view is a visual representation of the [Part](/parts) used in the harness. In this view you can add [Parts](/parts), modify their properties and see the how many times they are used in the harness. This is used to generate the <IconTable/> `BOM`.

<br />
<img src="/views/parts.png" width={800} style={{ borderRadius: "8px" }} />

## Table

This is the list of all your [Parts](/parts) in the harness.

`right-click` anywhere on the canvas to add a [Part](/parts) or click the <IconPlus/> `Add part` button.

`left-double-click` a parts value to edit it.

`left-double-click` a parts icon or click the <IconEdit/> button to edit the part's properties. [More Info](/parts#editing-parts)

See [Keyboard Shortcuts & Mouse Gestures](/shortcuts) for how to navigate the **Parts** view.
