# Main Menu

To open the **Main Menu** click on the <IconMenu2/> icon in the top left corner of the page.

<br />
<ul
  style={{
    listStyleType: "none",
    paddingLeft: 0,
    display: "flex",
    flexDirection: "column",
    gap: "0.5rem"
  }}
>
  <li><IconFile/> **New harness** - Creates a new harness under your account</li>
  <li><IconFolder/> [Open](/views/main-menu/open) - Opens the Open Harness panel</li>
  <li><IconFiles/> **Duplicate** - Duplicates the current harness</li>
  <li><IconFileExport/> [Export](/views/view-schematic) - Exports the current harness</li>
  <li><IconShare2/> [Share](/views/view-schematic) - Opens the Share panel</li>
  <li><IconTrash/> **Delete** - Deletes the current harness</li>
  <li><IconBook2/> [Guides](/) - Opens this documentation</li>
  <li><IconBrandDiscord/> [Discord](https://discord.com/invite/eTv6ph4Uzg) - Opens invite to discord server</li>
  <li><IconMessageReport/> **Give feedback** - Opens the feedback modal</li>
  <li><IconSettings/> [Settings](/views/main-menu/settings) - Opens the Settings panel</li>
  <li><IconUserCircle/> [Account](/views/view-schematic) - Opens the Account window</li>
  <li><IconLogout/> **Log out** - Logs out of the current account</li>
</ul>
